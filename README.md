# Bank Management System (Java)

A **Bank Management System** built using Java that simulates basic banking operations. This project demonstrates real-world banking functionalities and core Java concepts.

---

## Features

- Create and manage bank accounts  
- Deposit and withdraw money  
- Check account balance  
- Transaction handling  
- Basic validation and error handling  
- File handling / database integration (if applicable)

## Tech Stack

- **Language:** Java  
- **Concepts Used:**  
  - Object-Oriented Programming (OOP)  
  - File Handling  
  - Exception Handling  
  - Collections (if used)
